2): Question: Is there an age limit(Maximum or Minimum)?
Answer: Absolutely not! It's never too late or early to begin learning a new life skill. As long as you know how to use a computer and type on a keyboard.

Answer: It's up to you! By video chat we can screen share our code and you can see the tutors screen of code or digital whiteboard

Answer: Than you must have a parent or guardian sign you up for the tutoring service

Answer: We offer tutoring for up to third year college level Computer Science Courses

Answer: If you are already enrolled in a course than we can tutor you for that course. Else if you are self learning we can create a path and provide guidance and mentorship for whatever you want to create.

Answer: That depends on what you want to create. We teach languages for creating websites, apps, web automation, and games

8): Question: Do I have to buy any programs?
Answer: The top industry programming languages and editors are publically available for free. All you need is a computer with internet access. The only payment we ask of you is for our tutoring services.

9): Question: How much does this cost?
